Plugin Documentation

Goals available for this plugin:

Goal Report? Description
jira:create-new-jira-version No Goal that creates a version in a JIRA project.
jira:generate-release-notes No Goal that generates release notes based on a version in a JIRA project.
jira:jira-report Yes Creates a report of a JIRA version.
jira:mail-release-notes No Goal that generates release notes based on a version in a JIRA project.
jira:release-jira-version No Goal that creates a version in a JIRA project.
jira:transition-issues No Goal that made a transition on the given issues.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.5
JDK 1.7
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.github.lucapino</groupId>
          <artifactId>jira-maven-plugin</artifactId>
          <version>2.0.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.github.lucapino</groupId>
        <artifactId>jira-maven-plugin</artifactId>
        <version>2.0.0</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
  <!-- To use the report goals in your POM or parent POM -->
  <reporting>
    <plugins>
      <plugin>
        <groupId>com.github.lucapino</groupId>
        <artifactId>jira-maven-plugin</artifactId>
        <version>2.0.0</version>
      </plugin>
      ...
    </plugins>
  </reporting>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
